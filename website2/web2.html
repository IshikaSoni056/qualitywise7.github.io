Curriculum

Chart your success in 21 weeks of full stack development. Dive into product building, CV polishing, persuasive communication, and cutting-edge coding.

Week 1-4

Fundamentals of computer science, CV making, job opportunities

  • How to make impressive CV
  • How to find Job and where to find job
  • Understand Questions asked in interviews
  • Computer Science Basics
  • Psychometric Test - To check intelligence
  • Puzzles asked in interviews
  • Foundations of Computing: Exploring Binary Numbers, CPU Architecture, Storage Systems, Operating Systems, and More
  • Foundations of Computer Science: Data Structures, Design Principles, Algorithms, Networking, and more

Week 5-9

Mastering C Programming

  • Learn about C programming
  • Some Assignments
  • Exploring Basic Programs in C like pattern programs, sorting programs, and other programs
  • Introduction to Data Structures in C - array, linked list, stack and queue, tree, etc. in data structures
  • Interview Asked Problems - Preparation for the quetions and problems asked in interviews.

Week 10

English Communication

  • Learn english communication skills
  • Vocabulary and Practice guiding
  • Some assignments

Week 11-16

Web/frontend development

  • V8 Engine and DOM rendering, HTML, CSS
  • Basic to Advance JavaScript - Debounce function, async, callback functions, etc.
  • Modern Web Element Development - jQuery, practicing by making elements like caraosel, dropdown, calender, etc.
  • Frontend Project - Making a Project using all the technologies learned
  • Introduction to React Js - Virtual DOM, Routing, JSX, Hooks, etc.
  • ReactJs Project - Making a project using React Js

Week 17-21

Backend development

  • Introduction to Java - Core Java, java Syntax, OOPS, Control Flow, etc
  • Programming in core java
  • Advanced java - advance concepts of Java, Database modeling and SQL
  • Advance Java project - Creating a To-do app
  • Spring and spring boot - Dependency Injection, Spring Beans, Data Access with Spring, v, etc.
